# openbao-labs

This is a collection of hands-on labs for [OpenBao](https://openbao.org) running on the [Killercoda platform](https://killercoda.com).
Any changes to this repository are automatically reflected on Killercoda within a few minutes.

For initial repository onboarding, refer to [Killercoda documentation](https://killercoda.com/creators/get-started).
There are also docs available for [creating labs](https://killercoda.com/creators).

## Available Labs

- **[OpenBao Basics](./openbao-basics)**: Learn OpenBao basics.
- **[OpenBao Shamir Unseal](./openbao-manual-unseal)**: Learn about the OpenBao manual unsealing process with Shamir.
- **[OpenBao Auto Unseal](./openbao-auto-unseal)**: Learn about the OpenBao auto unsealing process.
- **[OpenBao Raft Snapshot](./openbao-snapshots)**: Learn about the OpenBao raft snapshot and restore process.

## Contributing

You are welcome to contribute and add your own labs. 

You can easily test the labs on your own account before submitting a pull request.

1. Fork the repository
2. Onboard your [forked repository on Killercoda](https://killercoda.com/creators/get-started)
3. Test your labs. :)
