In this scenario, we are going to configure OpenBao to use auto unseal.

When an OpenBao server start is in a sealed state. In this state, OpenBao is unable to decrypt it's storage. Before being able to interract with the OpenBao server, it needs to be unsealed so that the storage can be decrypted.

More information on the [Seal/Unseal mechanism and concept in the docs](https://openbao.org/docs/concepts/seal/#auto-unseal).
